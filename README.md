Student Management RESTful API (Node.js & Express.js)

A RESTful backend API built using Node.js and Express.js for managing student data. Supports CRUD operations and demonstrates backend development principles.

Project Overview

This project demonstrates how to create a RESTful API using Express.js. Users can perform Create, Read, Update, and Delete (CRUD) operations on student records. The API uses JSON for request and response and follows standard REST principles.

Features

Get all students (GET /students)

Get a single student by ID (GET /students/:id)

Add a new student (POST /students)

Update an existing student (PUT /students/:id)

Delete a student (DELETE /students/:id)

JSON-based request and response

Handles basic error responses (404, invalid data)

Technologies Used

Node.js – Server-side JavaScript runtime

Express.js – RESTful API framework

Body-parser – Parse JSON request bodies

Nodemon – Development server auto-restart
